Overview

We created a series of tools to help recommend restaurants to a user, based on data we pulled from Yelp. We applied various tools, including:

  • K-means clustering
  • Collaborative Filtering
  • Restricted Boltzmann Machine
  • Support Vector Machine

We use dataset from Yelp Dataset Challenge (https://www.yelp.com/dataset/challenge). We applied several methods, some of which we have learned in class, to make the prediction.

We started by filtering out entires that were no use to us. First, we only want businesses that are restaurants. We also don't want any insignificant data points, so we filtered out all businesses with less than XXX reviews. We ended up with a dataset of 17,153 users and 4,733 business. We then split the dataset, some of it for training and some of it for testing.
